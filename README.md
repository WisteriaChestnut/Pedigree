# Wisteria Chestnut Pedigree

> [!IMPORTANT]
> 本仓库收录的是**架空赛马、虚构血统与架空赛绩**。
> 页面会借用真实赛马、赛事、赛场及历史年份构建世界观，但其中的架空马匹、亲缘关系、参赛经历、名次、奖金与荣誉均不可作为现实赛马史料引用。

这是一个以静态 HTML 制作的架空纯血马谱系站。项目围绕 Ruffian 的虚构后代展开，通过六代血统表、近交标记、逐年赛绩与繁殖记录，构建一条嵌入真实美国赛马史的架空母系。

## 架空谱系

```text
Ruffian × Foolish Pleasure
└── Riot
    └── Rogue（父：Cigar）
        ├── Rigging（父：Day Trader）
        └── Ravager（父：Day Trader）
```

上述关系全部属于本项目的架空设定。真实马名只用于构建平行世界背景。

## 页面内容

各马匹页面通常包含：

- 多代血统表；
- 重复祖先与近交位置标记；
- 分年度赛绩、奖金及荣誉；
- 关键比赛的架空叙事；
- 后代与母系延续关系。

站点目前使用原生 HTML 与 CSS，不依赖构建工具或前端框架，可直接通过 GitHub Pages 部署。

## 真实性声明

本项目不是 JBIS、Equibase、The Jockey Club、PedigreeQuery 或其他赛马登记、赛果与血统机构的镜像或附属项目。

页面中的现实背景如需用于研究，请返回相应赛事组织、登记机构或权威数据库核验。转载、截图或二次创作时，请保留“架空 / Fictional”标识，避免内容脱离语境后被误认为史实。
